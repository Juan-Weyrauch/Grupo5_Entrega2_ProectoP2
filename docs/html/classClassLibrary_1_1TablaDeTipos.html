Detailed Description

La clase TablaDeTipos gestiona las relaciones de fortalezas, debilidades e inmunidades entre los tipos de Pokémon. Su función es proporcionar los datos sobre las interacciones de los tipos y calcular la efectividad de un ataque según el tipo del Pokémon atacado.
