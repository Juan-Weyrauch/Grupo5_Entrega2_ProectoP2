Detailed Description

Representa un ataque en el juego, con propiedades que definen su nombre, poder, precisión, tipo y atributos especiales. Esta clase implementa la interfaz IAtaque y se utiliza para definir los ataques que pueden ser utilizados por los Pokémon.
